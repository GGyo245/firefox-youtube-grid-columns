# AMO Listing Copy

## Add-on Name
- YouTube Grid Columns Controller

## Summary (KO)
- 유튜브 홈 그리드의 한 줄 영상 개수를 2~8개로 조절하고 쇼츠 선반을 숨기는 확장.

## Summary (EN)
- Control YouTube home grid columns from 2 to 8 videos per row and hide Shorts shelves.

## Description (KO)
유튜브 홈 화면에서 한 줄에 보이는 영상 개수를 원하는 값으로 바꿀 수 있습니다.

주요 기능:
- 한 줄 영상 개수 2~8개 조절
- 팝업 슬라이더로 즉시 적용
- 설정 자동 저장
- 쇼츠 선반 섹션 숨김

권한 안내:
- storage: 열 개수 설정 저장
- youtube.com 탭 접근: 유튜브 페이지 레이아웃 적용

개인정보:
- 데이터 수집 없음
- 외부 서버 통신 없음

## Description (EN)
Adjust how many videos appear in each row on YouTube home pages.

Features:
- Set 2 to 8 columns
- Apply instantly from popup slider
- Setting is saved automatically
- Shorts shelves are hidden

Permissions:
- storage: save column setting
- youtube.com tab access: apply layout on YouTube pages

Privacy:
- No data collection
- No external network requests

## Category Suggestion
- Shopping / Search Tools / Appearance
